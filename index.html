Consola - Sistema de incidentes

Desarrollo de una aplicación por consola para gestionar incidentes, utilizando Java y la tecnología JPA Hibernate para una eficiente interacción con bases de datos. El sistema permite realizar operaciones como agregar, visualizar, modificar y eliminar información de incidentes, técnicos, diagnósticos, soluciones, clientes y servicios. Utilicé patrones de diseño como State para manejar los estados de los incidentes (abierto, en preparación, cerrado) y Singleton para asegurar una única instancia de ciertos componentes. Además, implementé funciones lambda para generar reportes detallados, como rendimiento por técnico y análisis diarios y mensuales. Repositorio Website

Machine

Reballing | Reflow | Assembler | Electronic

Fabricacion de cero de maquina para soldar componentes SMD BGA

Ante la abundancia de plaquetas electronicas que requerian un cambio o un resoldado especial de componentes, nos encontramos con un problema y cree una solucion. Usando lenguaje Assembler programe una secuencia de comandos en un Pic 16f84, de la marca Microchip, para que caliente una Lampara especial por infrarrojo y que pueda realizar Reballing/Rework en placas con BGA y SMD. A partir de elementos reciclados y tiempo, usando una balanza vieja de chasis de acero , se fabricó un precalentador en la base con una resistencia eléctrica y una lámpara infraroja de alta temperatura que atacaba desde arriba una vez transcurridos los dos minutos de que esta la precalentaba. También tenía un control de sube y baja la lámpara por medio de un motor y engranajes sin fin que subia hasta arriba la misma para poder manipular la pieza a soldar. Repositorio Website
